PerfMonitor is a lightweight native macOS app that gives you a live view of your Mac's performance.
The clean interface tracks CPU usage (overall and per-core), memory usage with a full breakdown, and real-time CPU and GPU temperatures read directly from the system's sensors
Designed to be as light as possible, it idles at roughly 4% CPU(Looking to get this as low as possible, from what I have seen maybe 2-3 is realistic sitting on desktop),
and under 40 MB of RAM, so you can keep an eye on your system without weighing it down.
